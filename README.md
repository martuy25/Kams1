# Kams1
iHeartMedia Hack-A-Thon Project Files

KAMS-1 – The “Keep Alive Monitor” is designed to monitor standalone desk-top computers and immediately notify an
operator, 24/7, in the event the machine “freezes” or completes its 10 hour run utilizing a Raspberry Pi for hardware, python for software and Twilio for SMS messaging. - iHeart’s Research Division contained multiple machines running confidential software off network. The output 
from these machines were vital to the company and our device was designed to facilitate 24/7 monitoring
